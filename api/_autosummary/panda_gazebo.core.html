panda_gazebo.core¶

Module that contains the core components (classes and functions) that are needed for creating the panda-gazebo simulation.

Modules

panda_gazebo.core.control_server

This server is responsible for controlling the Panda arm/hand.

panda_gazebo.core.control_switcher

This class is responsible for switching the control type that is used for controlling the Panda Robot robot arm and hand.

panda_gazebo.core.group_publisher

Class used to group a number of publishers together.

panda_gazebo.core.moveit_server

A ros server that creates several of MoveIt services which can be used to control the Panda robot or retrieve sensor data for the robot.
